In your virtual machine, as the `root` user, navigate to the `/var/www/html` directory and clone the `Unitmaster` repository
<Note>
If the `html` directory is not present, proceed to create one; this will require `root` privilleges
</Note>
```bash
# Switch to the root user
sudo -i

# Switch to the /var/www directory
cd /var/www/html
```
### 1. Clone the repository
Clone the unitmaster_portal repository
```bash
  git clone <UNITMASTER_PORTAL-REPO-URL-HERE>
  ```
Switch to the `new_redesign_ke` branch
```bash
git checkout -b new_redesign_ke
```

### 3. Install project dependencies
```bash
composer install --no-scripts
```

### 4. Verify project directories
Ensure that the following directories are present in the project
  - `storage/framework`
  - `storage/framework/sessions`
  - `storage/framework/views`
  - `storage/framework/cache`
  - `bootstrap/cache`

### 5. Change ownership of the project
Change the ownership of the project folder to point to the `www-data` user (Nginx)
```bash
  chown -R www-data:www-data /path/to/unitmaster-portal-repo
  
  # For example
  # chown -R www-data:www-data /var/www/html/unitmaster_portal
  ```
