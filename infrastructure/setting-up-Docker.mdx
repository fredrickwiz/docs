## Install Docker
### Step 1: Update the package index
```bash
sudo apt update
```

### Step 2: Install the necessary packages for Docker's repository
```bash
sudo apt install -y ca-certificates curl gnupg
```

### Step 3: Add Docker's official GPG key.
```bash
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg
```

### Step 4: Add the Docker repository to APT sources:
```bash
echo \
    "deb [arch="$(dpkg --print-architecture)" signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
    "$(. /etc/os-release && echo "$VERSION_CODENAME")" stable" | \
    sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
```

### Step 5: Update the package index again with the new repository:
```bash
sudo apt update
```

### Step 6: Install Docker Engine, CLI, Containerd, and Compose plugins:
```bash
sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

### Step 7: Add your user to the `docker` group (optional but recommended for running Docker without `sudo`):
```bash
sudo usermod -aG docker $USER
```

### Step 8: You will need to log out and log back in for this change to take effect. Verify the installation.
```bash
docker run hello-world
```
<Note>
This command should download a test image and run a container, printing a "Hello from Docker!" message.
</Note>
