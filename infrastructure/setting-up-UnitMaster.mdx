In your virtual machine, as the `root` user, navigate to the `/var/www` directory and clone the `unitmaster` repository
```bash
# Switch to the root user
sudo -i

# Switch to the /var/www directory
cd /var/www
```
### 1. Clone the repository
```bash
  git clone <UNITMASTER-REPO-URL-HERE>
```
  Switch to the `develop-laravel-upgrade` branch
  ```bash
    git checkout -b develop-laravel-upgrade
    ```

### 2. Configure the environment variables
Create/update the `.env` file at the root of the project. Fill in the variables in the file according to your configurations.

### 3. Install project dependencies
These dependencies are essential for the project to run as expected
```bash
composer install --no-scripts
```

### 4. Verify project directories
Ensure that the following directories are present in the project
  - `storage/framework`
  - `storage/framework/sessions`
  - `storage/framework/views`
  - `storage/framework/cache`
  - `bootstrap/cache`

### 5. Change ownership of the project
Change the ownership of the project folder to point to the `www-data` user (Nginx)
```bash
  chown -R www-data:www-data /path/to/unitmaster_repo
  
  # For example
  # chown -R www-data:www-data /var/www/unitmasterv2
  ```
