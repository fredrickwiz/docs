In the Mamori PAM interface, navigate to the Wireguard on the navigation menu and click on `Configuration`
You will notice this text on the page `Wireguard has not yet been configured. Press This button to set the requried options.`

![Wireguard-Configuration-1](/images/pam/wireguard-config-1.png)

Click on the orange button to configure Wireguard
This will open a new window for you to fill in with the necessary info
![Wireguard-Configuration-2](/images/pam/wireguard-config-2.png)

<Note>Remember to set `Max Number of User Added Peers`, `Private IP` and `Private IP Subnet mask`</Note>

Once you're done, you can hit on `Save` and proceed to add `Peers`

On the `Peers` page, add your peers by clicking on the `+` icon at the top left
![Wireguard-add-peers](/images/pam/wireguard-add-peers.png)

You can use this pop up window to add a peer. You can even add a peer from an external directory
![Wireguard-add-peers-2](/images/pam/wireguard-add-peers-2.png)

Your peers will now be listed on this page
![Wireguard-add-peers-3](/images/pam/wireguard-add-peers-3.png)
